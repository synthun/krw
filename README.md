# krw - kais roblox wrapper

shell wrapper i decided to make to practice shell

## installation

```
$ make mime
$ sudo make install
```

run `krw` for usage

## dependencies

most of these are probably already installed on your system

+ `sh`
+ `make`
+ `curl`
+ `unzip`
+ `wine`
+ `wine-32bit`
+ `xdg-mime`
